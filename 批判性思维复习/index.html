批判性思维复习

第一章 绪论

批判性思维是合理的、反思性的思维，其目的在于决定我们的信念和行动。

如果总结为两个字：理性。总结为六个字：求真、论证、辩证

批判性思维过程的四个特有的原则:

  1. 发现和质疑基础假设。
  2. 检查事实的准确性和理论概念的逻辑一致性。
  3. 注重背景和具体情况的重要性。
  4. 想象和开创替代选择。

创新思维是发散性思维和收敛性思维的交互作用。

学与思的平衡是实现创新的关键机制。

批判性思维与收敛性思维和发散性思维的关系:
当我们发现和质疑基础假设，谨慎地检查事实的准确性和逻辑一致性,发掘隐含假设，追求完整论证的时候，批判性思维更多表现为收敛性思维。
当我们在训练学生的想象力，开创替代方案的时候，批判性思维更多表现为发散性思维。

批判性思维是创新思维的前提和基础，是高质量创新的保障,是创新思维的最核心内容。

第二章 理性

批判性思维的精髓：审慎求证，独立思考，倾听对方，自我批评，互动讨论启发，精益求精。总而言之就是理性。

审慎求证是理性的最重要、最根本特征。

理性就是讲道理,而且要讲好的道理。

好的道理由两部分构成:
1.好的理由:
必须是客观的、具体的、全面的、充分的。
2.好的论证:
谨慎地得出合适的结论。

论证是一组陈述句（或者断言），其中的一个陈述（或者断言）是根据其它陈述做出的。我们将它们分别称为结论和前提.
论证的基本构成:前提、结论和它们之间的推理关系:
前提→结论（共三项）

一个好的论证要符合两个条件:

  1. 前提的条件：前提为真或者可以接受
  2. 推理的条件：有效或者充分

论证的构成要素：

论点和背景、观点和立场、目的、事实根据、假定、语言和概念、解释和推理、含义和后果

第三章 阅读与论证

理解是阅读的基本目标

理解不仅是读懂作者表达的意思，还要回答为什么:他为什么这样说？就是发现作者的论证。

论证的标准化表达

基本的论证结构

  1. 单前提结构
  2. 多前提结构
    • 独立的多前提结构
    • 相互依赖的多前提结构
  3. 链式结构

图尔明论证模型

一个好论证由六个部分组成：

  1. 数据（Data）：用来论证的事实证据、理由。
  2. 保证（Warrant）：用来连接证据和结论之间的普遍性原则、假设或关系陈述，是推理的根据
  3. 断言（Claim）：要被证明的结论、主题、观点。
  4. 支撑（Backing）：用来支持保证（大前提）的陈述理由。
  5. 辩驳（Rebuttals） ：对已知反例、例外的考虑和反驳。
  6. 限定（Qualifiers）：对结论的范围和强度进行限定，常来自于对反例反驳的考虑。

第四章 澄清概念

概念出错的八种方式

  1. 概念模糊
  2. 偷换概念
  3. 同义反复
  4. 说套话
  5. 脱离具体环境
  6. 虚妄的定义精确
  7. 过度的学究
  8. 不恰当的定义

第五章 考察证据

“好论证”/“完满论证”的第一个基本标准是前提为真，或至少可以接受。
证据/理由为真是论证的第一道安全检验门。

产生虚假的原因：

  1. 名利
  2. 意识形态
  3. 情感偏见
  4. 认识的局限性
  5. 思维定式
  6. 观察渗透理论
  7. 传言的扭曲

第六章 推理

好论证的第二个标准：推理应该应该是相关的、有效的、充足的

六种论证出错的方式

  1. 无关谬误
  2. 不具体，不细致，用一种原因或原则来囊括一切
  3. 理由不充分、不重要、不全面
  4. 推理无效
  5. 类比无效
  6. 引用权威不适合

推理主要有两大类：演绎和归纳

归纳：从个别的,过去的例子推论到普遍的、未来的论断。

演绎：普遍到个别。

归纳是可能性的推理，演绎是必然性的推理

好的归纳是高概率的推理

演绎推理

三段论：大前提，小前提和结论

演绎推理的有效性：前提为真必然保证结论为真

前提为假，结论不一定为假，而是可真可假

演绎推理的正确格式

  1. 相关性问题
  2. 发现前提真和结论假的可能性——找反例

形式分析方法

五种有效的演绎推理形式

  1. 肯定前件的推理

  2. 否定后件推理（逆否）

  3. 选言推理

  4. 假言推理

  5. 三段式

两种无效的演绎推理形式

  1. 否定前件
  2. 肯定后件

归纳推理

高概率和内容成反比关系，信息内容越多，为真的概率越小

科学追求知识的增长：内容和概率的共同进步（大胆假设，小心求证）

归纳论证的评价

  1. 前提可靠的标准
  2. 前提对结论的充足支持的标准
    • 相关性
    • 证据的数量、种类、质量和结论的强度

统计推理

统计推理的评价

  1. 样本的大小
  2. 样本的代表性
  3. 调查方式

类比推理

最佳解释推理

实践推理

​ 实践推理以蕴含在事实中的逻辑为依据，是应用性学科的主要推理模式，在各门专业性课程的教学，应用性科学和即使工程研究中具有广泛的应用价值。

第七章 科学推理

因果关系是具有普遍性的导致关系

因果推理是发现或确认一种关系是不是因果关系。

因果关系的前提：两个事件同时/相继发生

因果推理的结论：它们是否有因果联系

因果推理的评价

  1. 需要说明因果机制
  2. 比较假说/选择最佳原因

科学发现包含发明假说和判别假说两大阶段

假说—演绎推理

假说—预言—验证—证伪在逻辑上绝对有效（否定后件）

  • 考虑初始条件（预言能发生要依赖的条件）和辅助假设（确定这些条件在检验中实现了，其他因素被控制、被排除了等等的假定)
  • 如果预言P没成功，否定的是（假说H、初始条件I和辅助假设A）的组合

假说—预言—验证—证实在逻辑上无效（肯定后件）

第八章 挖掘隐含假设

隐含支撑假设：对隐含前提的支撑

隐含深层支撑假设：对隐含支撑假设的支撑

隐含假设的类型

  1. 预定假设（指称假定）
  2. 隐含前提
  3. 支撑假设——前提的前提
  4. 含义：论证进一步蕴含的、不明显的观念和实际后果

挖掘隐含假设的目的：力图使论证变得完满。如果一直找不到满意度隐含假设来使论证完满，那么论证就是不完满的。

想象反例，构造解释性或限定性的隐含前提将反例排除

评价隐含前提的准则

  1. 忠实原则+宽容原则
  2. 强弱合适
  3. 不多余
  4. 可检验性和可信性
